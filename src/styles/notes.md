# Motion / 3D notes

- MacBook e celulares são modelos 3D feitos em CSS/DOM: camadas, profundidade, rotateX/rotateY, sombras e parallax.
- Interações: tilt no mouse, cursor glow, hover magnetic, cards com spotlight, gráficos animados e pills flutuantes.
- Para incluir modelos GLB reais depois, use `@react-three/fiber` + `@react-three/drei`, mantendo os componentes CSS como fallback.
